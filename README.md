# 地图基础功能封装(Openlayers)

## 运行测试环境

测试代码在`.test`文件夹中

**`Example`**

```
npm run dev
```

## 生成文档

**`Example`**

```
npm run doc
```

## 运行文档

**`Example`**
将`docs`文件夹下内容复制粘贴至`.docs`文件夹下，`注意`:docs文件夹下`readme.md`文件无需复制
```
cd .docs
// 进入.docs文件下执行如下命令
npm install
npm run dev
```

## 打包项目

**`Example`**

```
npm run build
```
